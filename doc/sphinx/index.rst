####################
ICOapi Documentation
####################

*******
Purpose
*******

A REST and WebSocket API using the Python `FastAPI library`_.

.. _FastAPI library: https://fastapi.tiangolo.com/

We currently support all major operating systems (Linux, macOS, Windows) on `CPython` ``3.12+``.

.. _CPython: https://en.wikipedia.org/wiki/CPython

When the API is running, it hosts an OpenAPI compliant documentation under ``/docs``, e.g. under http://localhost:33215/docs.
